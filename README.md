# ⚽ Goalkeeper & Set Piece Analysis Using API-SPORTS

This project demonstrates how Python can be used for sports analytics using the API-SPORTS platform. The focus is on football data, specifically goalkeeper performance and set piece analysis (corners, free kicks, and match events).

## 📊 Data Source

The project uses the API-SPORTS multi-sport REST API, which provides structured football data including fixtures, events, statistics, and player information.

🔗 API: https://www.api-sports.io/

## 📌 Objectives

- Extract football match data using a REST API
- Transform raw JSON into structured datasets
- Analyse goalkeeper performance metrics
- Explore set piece effectiveness (corners/free kicks)
- Demonstrate Python-based sports analytics workflow

## 🧠 Key Features

- Match event analysis (goals, cards, substitutions)
- Statistical breakdown (shots, possession, corners)
- Goalkeeper performance tracking (goals conceded, saves where available)
- Set piece analysis using corner and free kick indicators

## 🛠️ Technologies Used

- Python
- Pandas
- Requests
- API-SPORTS
